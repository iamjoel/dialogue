## 问题
对我来说，人生有 3 个重要的问题:  
1. [如何延长健康寿命？](../tech/health/readme.md)
2. [如何获取财富？](../tech/wealth/readme.md)
3. [如何找到人生的意义?](../tech/the-meaning-of-life/readme.md)。

健康是基础，财富是工具，意义是指引。三者相互支持，共同构建了一个充实而平衡的人生。
