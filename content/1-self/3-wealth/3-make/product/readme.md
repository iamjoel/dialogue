发现用户的需求。

## 我的洞见
* 做竞品分析的局限性：不能知道用户的真实需求。
  * 我们只知道竞品的功能，但获得不到用户的真实需求。只能靠功能去猜测，这是不靠谱的。自己做产品的时候，是可以知道用户的真实需求的(产品不是太糟糕，以及反馈渠道通畅的情况下)。
  * 追踪竞品的受欢迎程度(Star 数，付费)，也知道知道他们在运营上做对了什么。并不知道后续。比如，用户涨了很多，但留存很低，或付费很差。
  * 可以靠看用户对竞品的评价，来稍微弥补下这个缺点。
