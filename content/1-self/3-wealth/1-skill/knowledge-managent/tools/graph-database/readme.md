---
title: 双链笔记
---
双链笔记标明了笔记(概念)之间的关系。方便了解概念之间的关系和检索信息。依据网络的六度分隔理论，只要我们对笔记多建立关联，哪怕你的数据库中有几十亿条笔记，也能通过中间的6条笔记找到你要找的那条回忆不起来的笔记。

最早支持双链笔记的是 [Roam](../roam.md)。
