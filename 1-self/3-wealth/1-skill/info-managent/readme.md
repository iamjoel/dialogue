## 管理工具

## 信息记忆
为了[记忆](../memory/readme.md)，就是要经常回顾(记忆曲线)。有的会用 Flash Card。在 VSCode 可以用 [Recall](../../2-tool/coding/ide/vscode/plugins/recall.md)。一些笔记软件支持，如：[Logseq](./tools/logseq.md),[RemNote](https://www.remnote.com/)(面向学生)。
